Place skill cutscene videos here.

Supported names used by the game:
- skill_p1.mp4 (or skill_p1.webm)
- skill_p2.mp4 (or skill_p2.webm)

In-game playback uses PNG frame sequences (no external player):
- videos/skill/p1/frame_0001.png ... frame_XXXX.png
- videos/skill/p2/frame_0001.png ... frame_XXXX.png

If frame folders are missing, the game will show a missing cutscene message.
